# Elyassi Exchange 

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Create a `.env` file (you can copy from `.env.example`) and add SMTP credentials.
   - Set `PUBLIC_BASE_URL` to your site URL so email links work.

3. Start the server:
   ```bash
   npm run dev
   ```

4. Open `http://localhost:3000`.
